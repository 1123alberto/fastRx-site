# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-08

### Added
- Initial release of the bilingual FastRx landing page (`el_GR` and `en_US`).
- Responsive UI built with HTML, CSS, and vanilla JS.
- Smooth language selector toggling between Greek and English.
- SEO and Open Graph metadata configuration for social sharing.
- Interactive privacy policy modal popup.

### Changed
- Refined Greek and English translations across the hero, workflow, status, and contact sections:
  - Updated hero subtitles for both locales to emphasize development status.
  - Refined English status main text to prioritize speed, clarity, and ultimate physician control.
  - Updated Greek clinical workflow step titles and descriptions (e.g., "Πρότυπο Συνταγής", "Έτοιμη για Υποβολή").
- Removed accents from Greek uppercase status badge to match standard typography rules.
